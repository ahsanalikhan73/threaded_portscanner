

Advanced Port Scanner
------------------------------
It will by default scans first 1000 ports
it will only scan single port with -p option
it will scan range of ports with -pr option



Multi-Threaded Port Scanner
-------------------------------

it will scan single and multiple ports (separated by comma) with -p option
it will scan range of ports with -pr option


Port Banners
--------------------------------

It will return Banners recieved from the open port

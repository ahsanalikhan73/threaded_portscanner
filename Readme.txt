
python3 Multi-Threaded Port Scanner.py --help

usage: Multi-Threaded Port Scanner.py [-h] [-t TARGET] [-p PORT] [-pr PR]

optional arguments:
  -h, --help            show this help message and exit
  -t TARGET, --target TARGET
                        Specify Host IP / Address
  -p PORT, --port PORT  Specify The Port To Scan
  -pr PR, --port-range PR
                        Specify The Range Of Ports To Scan


This script can scan single, multiple or range of ports you want but not at the same time.

Parameters:	
------------
	-t 	--> specifies your target
	-p 	--> scan single or multiple ports separated by comma(s) e.g, 80,111,443 etc
	-pr	--> scan range of ports separated by dash (-)  e.g., 20-200 etc
